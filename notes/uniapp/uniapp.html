    # 4.下载Android离线SDKAndroid离线SDK open in new window

    # 5.生成appkey的网站：DCLOUD的开发者中心开发者中心open in new window

    # 6.生成appkey需要我们去生成Android的签名文件

    # 7.生成Android的签名文件

    (1.)首先在官网下载JRE8open in new window

    (2.)解压完压缩包在文件夹的bin目录下打开终端 输入以下命令可以直接生成证书

    keytool -genkey -alias testalias -keyalg RSA -keysize 2048 -validity 36500 -keystore test.keystore

    (3.)根据自己个人习惯输入密钥

    (4.)根据命令可以查看具体的证书信息

    keytool -list -v -keystore test.keystore


    (4.)由于现在的jdk生成的签名文件中没有MD5，我们需要在jre的bin目录下，通过Git Bash here就可以查看证书的md5，test.keystore是我的证书名

    ./keytool -exportcert -keystore test.keystore | openssl dgst -md5

    # 8.以上步骤完成之后在开发者中心open in new window应用信息填写我们的Android签名然后生成我们索需要的appkey

    # 9.下载Android StudioAndroid Studioopen in new window

    # 10.把我们下载好的Android SDK离线包解压完之后把这个文件夹HBuilder-Integrate-AS在我们的Android Studio打开

    # 11.打开之后等待项目进行编译
